# Warehouse Management System

This project now uses English for project text and code comments.

Use [README_EN.md](README_EN.md) as the main project README.

Quick start:

```bash
gradle run
```

Main files:

- `warehouse.mas2j`: multi-agent system configuration.
- `src/agt/`: Jason agents.
- `src/env/warehouse/`: Java environment and model classes.
- `doc/MEMORIA_DRAFT_EN.md`: English report draft.
