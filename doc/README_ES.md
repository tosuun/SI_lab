# Delivery Instructions

This project now uses English for project text and code comments.

Use [README_EN.md](README_EN.md) for the full delivery instructions.

Required delivery items:

- `memoria.pdf` in this `doc/` folder.
- Complete source code.
- A ZIP file named `warehouse_grupoXX.zip`.

Before delivery, check that:

- The system runs with `gradle run`.
- The robots move containers.
- The scheduler starts output cycles.
- The supervisor prints no-space and missed-deadline events.
- Robot delivery events use the required `EVENT | ...` format.
